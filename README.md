# LocalServe

LocalServe is a local service platform that connects customers with trusted local professionals for various services like cleaning, plumbing, painting, and more.

## Features
- Browse and filter services by category, location, and price.
- Book services with a seamless 4-step workflow including a dummy payment step.
- Unified platform for both Clients and Providers.
- Built with React, Vite, Tailwind CSS, and Supabase.
